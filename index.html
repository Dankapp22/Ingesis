    Creación, ejecución y mantenimiento de sistema de ventas de tickets web.

    Almacenamiento de la información en bases de datos, exportación de dicha
    información, implementación de sistema de venta (boleta/factura) para la empresa.

    Creación, ejecución y mantenimiento de sistema para ventas de ticket.
